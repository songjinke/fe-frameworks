# FE Frameworks
This is a project to show examples of different FE frameworks.

## Setup
```bash
# install dependencies
npm install

# serve in dev mode
npm start
```

## Environments
1. Angular application: http://localhost:4200/
2. React application: http://localhost:3000/
3. Vue application: http://localhost:8080/
